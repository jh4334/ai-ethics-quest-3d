# 2막 「잡음의 군도」 — 확장 기획서 (Fable)

> 상태: 기획 확정 대기. 공모전 제출본(1막 = 현재 섬)은 이 확장과 무관하게 안정 유지한다.
> 확장 착수는 제출본 태그 이후. 모든 확장은 현행 사양(외부 에셋 0 · 결정성 · 저사양 · 무설치 웹)을 유지한다.

## 1. 문제 진단 — "아이템이 매력이 없다"의 근본 원인

젤다 아이템의 매력은 **동사(verb)** 에서 나온다. 훅샷은 "끌어당긴다", 폭탄은 "부순다" —
아이템이 세계와 상호작용하는 새로운 방법을 열어 준다.

지금 우리의 약속 도구 4종은 **열쇠**다. 관문 앞에서 자동으로 쓰이고, 보스전에서 "맞는 것을
골라 드는" 대상일 뿐, 플레이어가 **버튼을 눌러 발동하는 동사가 없다**. 그래서 방탈출처럼
1회성으로 소비되고, 던전도 재방문 가치가 없다. 스토리가 아니라 **도구 체계**가 한계였다.

## 2. 해법 — 도구 4종에 '동사'를 부여한다

기존 윤리 정체성은 유지하고, 각 도구에 상시 발동 가능한 동사를 더한다.
(모바일: 우측 도구 버튼 탭 = 동사 발동, A = 상호작용. 키보드: Q/1~4 선택 + F 발동.)

| 도구 | 윤리 의미 | 동사 | 월드/던전 활용 | 전투 활용 |
|---|---|---|---|---|
| 🛡️ 약속의 방패 (동의) | 허락 없인 못 지나간다 | **막기·밀쳐내기** | 잡음 덩굴을 밀어 길 내기, 바람·급류 버티기 | 잡음 파도를 타이밍 가드로 튕겨 되돌리기 |
| 🧭 진실의 나침반 (검증) | 진실을 끌어온다 | **끌어당기기** (훅샷/자석) | 멀리 있는 상자·구슬을 당기고, 진실 앵커로 몸을 끌어 이동(간극 건너기) | 도망가는 노이즈 조각을 붙잡기 |
| 🔔 출처의 종 (출처) | 소리가 가짜를 흩는다 | **울림 충격파** (비폭력 '폭탄') | 잡음 벽·가짜 덩어리를 울려서 흩어버리기, 숨은 출처 드러내기 | 주변 투사체 일소(쿨다운) |
| 🪞 다양성의 거울 (다양성) | 다른 관점을 비춘다 | **빛 반사·정체 드러내기** | 들고 다니는 반사면 — 빔 퍼즐 어디서나, 환영(가짜 지형) 정체 밝히기 | 보스의 가짜 분신 구별 |

**게이팅 원칙(메트로배니아)**: 스테이지 N의 던전은 이전 스테이지 도구의 동사를 요구한다.
새 도구를 얻으면 이전 섬의 숨은 방이 열린다 → 재방문 가치.

## 3. 세계관 2막 — 잡음의 군도 (프롤로그 + 5 스테이지)

**서사 골격 — "압도적 패배 → 동사 각성 → 재대결 승리"**

현재 1막 전체(4사당 + 도구 획득)는 확장판에서 **프롤로그**가 된다. 프롤로그 끝의
노이즈 첫 대결은 **이길 수 없다**: 도구가 전부 튕겨나가는 압도적 패배 이벤트(3회
공격 시도 후 스크립트 진행 — 게임오버가 아니라 연출). 노이즈는 삼켰던 **기억 파편**을
흩뿌리며 바다 건너로 도망치고, 파편을 삼킨 네 섬의 **수호 정령들이 같은 병을 앓기
시작한다.** 도트: "도구를 *갖고 있는 것*과 *쓸 줄 아는 것*은 달라 — 진짜 힘(동사)을
깨우자." 섬을 돌며 동사를 각성하고 정령들을 치료한 뒤, 마지막 섬에서 노이즈와
**재대결해 최종 승리**(제압 → 가르침 → 노바 재탄생 → 증명서, 기존 엔딩이 5막으로 이동).

패배가 벌이 아니라 성장의 시작이라는 점에서 1막의 회복 서사(틀려도 다시 도전)와
정확히 같은 교훈을 게임 구조 전체로 확장한다.

교육용 분량 원칙: **총 5스테이지, 스테이지당 10–15분** (수업 1차시에 1스테이지).
주제는 짝지어 합치고, 각 섬은 **던전 2개 + 정령 보스 1**로 밀도를 채운다.
각 스테이지는 **그 섬에서 깨어난 동사를 주력으로 사용**하도록 설계한다.

### 스테이지 구성 (초등 5–6 · 2022 개정 실과/도덕 연계)

| # | 섬 | 주제 묶음 (교육 프레임) | 깨어나는 동사 | 콘텐츠 |
|---|---|---|---|---|
| P | 시작의 섬 (현재 1막) | 개인정보·편향·저작권·딥페이크 — "AI 윤리의 네 약속" | 도구 4종 획득 (동사는 아직 잠듦) | 던전 4 + **노이즈 패배 이벤트** (≈15–20분) |
| 1 | 속삭임 곶 | 악플·혐오표현 + 디지털 발자국 — **"내가 남기는 것"** | 🛡️ 방패: 막기·밀쳐내기 | 던전 「말-화살 회랑」(가드로 화살을 주인에게 되돌리기) + 「발자국 갯벌」(지워지지 않는 흔적 되짚기) + 정령: 상처 입은 바닷새 |
| 2 | 메아리 동굴 | 가짜뉴스·출처 + 필터버블·알고리즘 추천 — **"내게 들어오는 것"** | 🔔 종: 울림 충격파 · 🪞 거울: 정체 드러내기 | 던전 「소문의 벽」(충격파로 흩고 원본 찾기) + 「거울 미궁」(같은 상만 비추는 방을 다른 관점으로 탈출) + 정령: 메아리에 갇힌 고래 |
| 3 | 모래시계 항구 | 스크린타임·디지털 웰빙 + AI 생성물 표시·정직한 사용 — **"AI와 나의 습관"** | 🧭 나침반: 끌어당기기 | 던전 「모래시계 사구」(멈출 때를 아는 타이밍 퍼즐) + 「표시 없는 화물선」(생성물 표시 매칭 심화) + 정령: 잠들지 못하는 등대거북 |
| 4 | 기억의 심장 (외곽) | 종합 + 인간-AI 협업 — **"함께 살아가기"** | 4동사 조합 훈련 | 3구역 연속 던전(구역마다 다른 동사 필수) + 노바와 역할 분담 협동 구간 |
| 5 | 기억의 심장 (심부) | 최종 재대결 | 4동사 총동원 | **노이즈 리매치** — 페이즈마다 다른 동사가 열쇠(방패로 파도 반사→나침반으로 붙잡기→종으로 껍질 흩기→거울로 진짜 몸 찾기) → 제압 → 가르침 → 노바 → 증명서 (≈15분) |

### M6 확정 사항 — 패배 이벤트의 위치 (2026-07 재결정)

- **프롤로그 승리·증명서는 유지한다.** 증명서·학습 리포트는 수업 1차시 구조와 제출물의
  핵심 신호라, 심부 승리 뒤로 옮기면 지도안·활동지 개편 + 불가침 신호 대수술이 된다
  (마감 3주 전 고위험). 원안(프롤로그 무적화)은 기각이 아니라 **위치 이동**이다.
- **"압도적 패배 → 동사 각성 → 승리" 서사는 심부 리매치 도입부로 옮긴다**:
  심부에서 노이즈의 잔영과 마주치면 도구가 전부 튕겨나가는 패배 연출(스크립트)
  → 치유한 정령 셋이 달려와 동사의 힘을 완전히 깨움 → 페이즈별 동사전 → 승리
  → 2막 엔딩(군도 완전 치유). 같은 교훈을 위험 없이 완성한다.
- M6 슬라이스: **S1 외곽**(4봉인 동사 조합 훈련 + 심부 개방) → **S2 심부**(패배 연출
  + 잔영 리매치 + 2막 엔딩) → S3 문서·지도안 동기화.

### 공모전 제출본과의 관계 (확정: 단일 트랙)

- 마감 7월 말 · 심사 10월. 브랜치 분리 없이 **main 단일 트랙**으로 확장한다.
  롤백 지점은 태그 `v1.0-act1`(= 1막 완결 커밋 `445bff2`).
- **순서 원칙**: 배포본은 항상 처음부터 끝까지 플레이 가능해야 한다. (M3~M5는 이 원칙대로
  프롤로그 승리 흐름을 유지한 채 확장을 얹었고, M6도 같다 — 위 재결정 참조.)
- **프롤로그 시네마틱**(M1-S3): 인엔진 연출(카메라 플라이오버 + 레터박스 + 자막 카드
  + 절차 음악, 스킵 가능) — 외부 영상 파일 없이 스토리 도입을 강화한다.

### 스테이지당 10–15분 해부 (섬 1–3 공통 뼈대)

1. **도착 서사** (≈1분): 뗏목 상륙, 병든 정령의 증상을 눈으로 보여줌 (대화 최소)
2. **섬 탐험** (2–3분): 동사 각성 제단 찾기 + 주제 상호작용 오브젝트 2–3개
3. **던전 A** (3–4분): 새 동사의 기본 사용법을 익히는 퍼즐
4. **던전 B** (3–4분): 동사 응용 + 이전 섬 동사와 조합
5. **정령 치료전** (2–3분): 그 섬 동사가 열쇠인 패턴 보스 — 치료 후 본모습 회복
6. **마무리** (≈1분): 정령의 한 줄 교훈 + 학습 리포트 기록 + 다음 섬 항로 개방

정령 보스는 섬마다 1종(병든 모습 → 치료 후 본모습), 패턴은 그 섬의 동사로만 풀린다.

## 4. 기술 아키텍처 (현행 사양 그대로 확장)

- **데이터 주도**: `DUNGEON_ROOMS` 패턴을 `STAGES`로 승격 — 섬 지형·던전·정령·항해 좌표까지
  순수 데이터 + 순수 로직(`stageData.js`/`toolVerbs.js`, THREE 무의존, node 테스트).
- **씬 수명주기**: 검증된 던전 패턴 재사용 — 스테이지 lazy build + dispose, 오버월드 Group 스왑.
  항해는 가벼운 바다 씬 1개(섬 실루엣 스프라이트)로.
- **도구 동사 시스템**: `useToolVerb(toolId, game)` 단일 진입점, 던전·오버월드·전투 공용.
  쿨다운·사거리 전부 상수(결정성).
- **세이브 v2**: `progress.stages[stageId]` 맵 추가, `normalizeProgress`가 v1 세이브를
  스테이지 1 완료 상태로 마이그레이션. 학습 리포트는 스테이지별 섹션으로 확장(기존 신호 보존).
- **성능 예산**: 동시 로드 씬 1개 원칙 유지 → 섬이 10개여도 상주 메모리는 지금과 동일 수준.
  e2e에 스테이지별 시나리오 추가.

## 6. 리스크와 원칙

- **공모전 우선**: 마감 전에는 M1도 착수하지 않는다. 제출본의 안정성이 최우선.
- 동사 추가는 기존 퍼즐을 깨지 않게 — 1막 던전은 "동사 없이도 풀리고, 동사가 있으면 더 빨리".
- 주제·문구는 지도안과 함께 갱신(문서-게임 정합성은 e2e급 체크리스트로 관리).
