# AI 윤리 게임 3부작 — 「AI 윤리 수호자의 여정」 통합 기획서

> 교육자료전·AI 활용 공모전 출품용 통합 패키지 개요.
> 대상: 초등학교 5~6학년 (3~4학년 연계 가능) · 교과: 도덕·실과(정보)·국어(매체) + 창의적 체험활동

## 1. 기획 의도 — 왜 '게임 3부작'인가

2022 개정 교육과정은 **디지털 소양을 언어·수리 소양과 함께 3대 기초소양**으로 격상하고,
실과에 '디지털 사회와 인공지능' 영역을, 국어에 '매체' 영역을 신설했으며, 도덕에는
인공지능 로봇과의 관계([6도02-03]), 정보통신 윤리([4도03-02]) 성취기준을 명시했다.
AI 윤리는 2022 개정의 최대 화두 중 하나이지만, 현장에는 **강의식 자료·영상 자료가 대부분이고
학생이 직접 판단하고 결과를 겪어 보는 게임형 자료는 공백**에 가깝다.

본 3부작은 이 공백을 겨냥한다. 세 게임은 각각 독립적으로 수업에 쓸 수 있지만,
**"개념 탐험 → 3D 심화 적용 → 역량 진단·성찰"**의 학습 사이클로 이어질 때
지식(알기) - 판단(해보기) - 태도(돌아보기)를 관통하는 완결된 프로그램이 된다.

## 3. 학습 여정 설계 (총 6차시)

```
[1부] 1~2차시          [2부] 3~4차시            [3부] 5차시         [종합] 6차시
개념 탐험·퀴즈    →    3D 섬에서 4대 쟁점    →   15개 실전 미션   →   카드뉴스 제작
"무엇이 문제인가"      판단·토론 "나라면?"        역량 진단·성찰        캠페인·발표
```

- **1부(배움)**: 도트 맵을 탐험하며 개인정보·저작권·가짜정보·생성형 AI·책임 개념을 퍼즐과 퀴즈로 익힌다. 위험 선택은 처벌이 아니라 회고 질문으로 이어진다.
- **2부(적용)**: 3D 섬의 네 구역(개인정보 마을·편향의 숲·저작권 유적·딥페이크 동굴)에서 NPC와 대화하고 사당 서브퀘스트로 '약속의 도구'를 얻어, 노이즈가 막은 관문의 윤리적 선택을 도구로 해결(실수하면 처벌 대신 회복 분기)하며 윤리 조각을 모은다. 최종장에서는 나쁜 데이터로 자란 빌런 노이즈를 지우지 않고 '다시 가르쳐' 노바로 되살리는 선택으로, AI가 잘못되는 원리와 바로잡는 법을 서사로 체득한다.
- **3부(진단·성찰)**: 사전 자기진단 → 실생활 딜레마 15미션(실제 사례 포함) → 4역량 리포트와 사후 진단으로 자신의 성장을 확인한다.
- **종합(표현)**: 배운 윤리를 복합양식 매체 자료(카드뉴스·포스터)로 제작·공유한다([6국06-03]).

상세 차시 계획: [`교수학습-지도안.md`](./교수학습-지도안.md) · 성취기준 근거: [`성취기준-매핑.md`](./성취기준-매핑.md) · 학생용: [`차시별-활동지.md`](./차시별-활동지.md) · 제출용: [`공모전-제출-요약본.md`](./공모전-제출-요약본.md)

## 5. 공모전 차별점

1. **게임형 자료 공백 공략**: AI 윤리를 '읽는' 자료가 아니라 '겪는' 자료로. 선택-결과-회고 루프가 태도 변화까지 다룬다.
2. **배움-적용-진단의 완결 구조**: 단일 게임이 아니라 학습 사이클 전체를 덮는 3부작. 차시별로 떼어 쓸 수도, 6차시 프로그램으로 통합 운영할 수도 있다.
3. **교사 즉시 활용성**: 성취기준 매핑 + 차시별 지도안 + 활동지 + 루브릭 + 교사 대시보드 동봉. 무설치·오프라인 동작으로 교실 인프라 제약이 없다.
4. **안전 설계**: 계정·서버·학생 데이터 수집 없음. 딥페이크 등 민감 소재는 가상 상황으로만 다룬다.
5. **증거 기반 수업**: 1부 진단 리포트와 3부 사전·사후 비교로 성장을 데이터로 확인하고 개별 지도에 연결한다.

## 7. 유의 사항

- 성취기준 원문은 교육부 고시 제2022-33호 별책 5(국어)·별책 6(도덕)·별책 10(실과) 기준으로 작성했다. **공모전 최종 제출 전 고시 원문과 한 번 더 대조**를 권장한다.
- 제출 양식에 개인정보(이름·소속)를 포함하지 않는다(심사 제외 사유가 되는 공모전이 많음).
- 게임은 개념 학습용 교육 자료이며, 실제 AI 서비스 이용 규칙은 학교·보호자 지침을 따르도록 지도한다.
